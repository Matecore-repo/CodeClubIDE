# Security Policy / Política de Seguridad

## Reporting a Vulnerability / Reportar una Vulnerabilidad

If you find a security vulnerability, please **do not open a public issue**. Send an email to **iangel.oned@gmail.com** instead.

Si encontrás una vulnerabilidad de seguridad, **no abras un issue público**. Envía un email a **iangel.oned@gmail.com**.

We aim to respond within 48 hours and will keep you updated on the progress.
Apuntamos a responder dentro de 48 horas y te mantendremos al tanto del progreso.

Thank you for helping keep Code Club IDE safe for everyone.
Gracias por ayudar a mantener Code Club IDE seguro para todos.
